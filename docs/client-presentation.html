Архитектура

9 производственных модулей

Каждый модуль покрыт юнит- и интеграционными тестами, реализован с учётом production-ready требований.

🔐
Authentication & Security
Полный цикл аутентификации: JWT RS256, ротация refresh-токенов, TOTP 2FA, bcrypt-хэширование паролей, blacklist ревокации.
JWT RS256 TOTP 2FA refresh rotation bcrypt
🏗️
App & Entity Builder
Динамическая метамодель: мульти-тенант приложения, сущности, 15+ типов полей, связи между сущностями.
multi-app metamodel 15+ field types relations
🗄️
Records Engine
JSONB-хранилище записей, cursor-пагинация, расширенная фильтрация (10+ операторов), мягкое удаление, версионирование.
JSONB cursor pagination soft delete 10+ operators
⚡
Rules Engine
Визуальное AST-дерево условий, 7 типов действий, определение циклических зависимостей, изолированное выполнение в Celery sandbox.
AST conditions 7 action types cycle detection sandbox
🔄
Workflow Engine
FSM конечные автоматы, оптимистичная блокировка (409 при race condition), SLA-таймеры через Celery ETA, полный лог переходов.
FSM optimistic lock SLA timers transition log
🎨
UI Builder
6 типов представлений (table/form/kanban/calendar/gallery/detail), конфигурация полей, page builder с блочной структурой.
6 view types kanban calendar page builder
🔗
Integration Layer
Outbox pattern для надёжной доставки, webhook-подписки, HMAC-SHA256 подпись, exponential backoff (60s/5m/30m), retry-очередь.
outbox pattern webhooks HMAC-SHA256 retry backoff
🛡️
Field-Level ABAC
Attribute-Based Access Control на уровне полей. Deny-wins семантика, per-role разрешения, автоматическая фильтрация при чтении/записи.
ABAC deny-wins per-role field filtering
📊
Production Readiness
Rate limiting (slowapi, JWT-bucketing), Prometheus counters по всем подсистемам, OpenTelemetry tracing, /health/live + /health/ready пробы.
rate limiting Prometheus OpenTelemetry health probes
